    Magic.h のダウンロード

    インストール方法について

    • Windowsの場合
      • ドキュメントフォルダにある Arduino -> libraries の中に、magic という名前のフォルダを作り、その中にダウンロードした zip ファイルの中身を置いてください。
      • Documents/Arduino/libraries/magic/
    • Mac OS Xの場合
      • 書類フォルダにある Arduino -> libraries の中に、magic という名前のフォルダを作り、その中にダウンロードした zip ファイルの中身を置いてください。

    また、 「Arduino ライブラリ インストール」などで検索してみてください。 (問い合わせ/質問等については今後検討します。)
